# PopX – Educase Assignment

PopX is a simple and clean frontend prototype that demonstrates a basic **signup, login, and account interface**.
This project focuses on building a smooth user experience using only frontend technologies.



## Live Demo - Deployed project Link

👉 https://ajay-educasepopx.netlify.app/


## 📁 Project Overview

This project includes the following pages:

* 🏠 **Home Page** – Entry point of the application
* 📝 **Signup Page** – User registration interface
* 🔐 **Login Page** – User authentication UI
* 👤 **Account Page** – Basic profile display

---

## How to Run the Project

### ✅ Option 1: Open Directly

Simply open `index.html` in your browser.

---

### ✅ Option 2: Run with Local Server (Recommended)

```bash
python -m http.server 8000
```

Then open:
👉 http://localhost:8000
